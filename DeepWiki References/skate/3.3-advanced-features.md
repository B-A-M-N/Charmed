Advanced List Command Options
-----------------------------

Skate provides several filtering and formatting options when listing key-value pairs.

### Filtering Output

Skate allows you to control what data is shown when listing your key-value store:

*   **Keys only mode**: `skate list --keys-only` or `skate list -k` Shows only the keys, useful for scripting or when you just need to see what's stored.
    
*   **Values only mode**: `skate list --values-only` or `skate list -v` Shows only the values, useful when keys are known or not needed.
    

### Formatting Output

You can control how the output is presented:

*   **Custom delimiter**: `skate list --delimiter "="` or `skate list -d "="` Changes the separator between keys and values (default is tab).
    
*   **Reverse order**: `skate list --reverse` or `skate list -r` Lists items in reverse lexicographic order.

Performance Considerations
--------------------------

When working with large amounts of data or binary files, keep in mind:

1.  The BadgerDB backend is efficient but has memory usage proportional to operation size
2.  Very large binary files are supported but may impact performance
3.  List operations fetch all keys by default, which can be memory-intensive for large databases

For large datasets, consider:

*   Using separate databases to partition data
*   Using the `--keys-only` flag when just checking available keys
*   Implementing pagination in your scripts when processing large datasets

Summary
-------

Skate provides numerous advanced features that extend its utility beyond simple key-value storage:

| Feature Category | Description | Example Usage |
| --- | --- | --- |
| Binary Data | Store and retrieve arbitrary binary data | `skate set image < photo.jpg` |
| List Options | Filter and format list output | `skate list -k -r` |
| Input Flexibility | Read from standard input or files | `cat file.txt \| skate set key` |
| Unicode Support | Full support for international characters | `skate set 日本語 "こんにちは"` |
| Script Integration | Use in scripts for various workflows | `token=$(skate get api_token)` |

These advanced features make Skate a versatile tool that can be integrated into various workflows and systems.
