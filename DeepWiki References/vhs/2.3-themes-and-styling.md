This page documents the theming and styling capabilities of VHS, which allow you to customize the visual appearance of your terminal recordings. This includes terminal color schemes (themes) as well as visual elements like window bars, rounded corners, and padding.

For information about output formats and video encoding options, see [Output Options](https://deepwiki.com/charmbracelet/vhs/2.2-output-options)
.

Theme System Overview
---------------------

VHS allows you to apply predefined color schemes to your terminal recordings through its theme system. You can also customize individual color properties or create your own themes.

Sources: [themes.go1-204](https://github.com/charmbracelet/vhs/blob/f28239f3/themes.go#L1-L204)

Using Themes
------------

VHS comes with over 340 pre-defined themes that you can apply to your terminal recordings. To use a theme in your tape file, use the `Set Theme` command followed by the theme name in quotes:

    Set Theme "Catppuccin Mocha"
    

You can view all available themes by running `vhs themes` in your terminal.

### Popular Themes

| Theme Name | Description | Dark/Light |
| --- | --- | --- |
| Catppuccin Mocha | A soothing pastel theme (dark variant) | Dark |
| Catppuccin Latte | A soothing pastel theme (light variant) | Light |
| Dracula | A dark theme with vibrant colors | Dark |
| Nord | A cold, bluish color palette | Dark |
| Solarized Dark | A popular dark theme with optimal contrast | Dark |
| Solarized Light | A popular light theme with optimal contrast | Light |
| Tokyo Night | A dark theme inspired by Tokyo at night | Dark |
| One Dark | Atom's iconic dark theme | Dark |

Sources: [themes.go140-158](https://github.com/charmbracelet/vhs/blob/f28239f3/themes.go#L140-L158)
 [THEMES.md1-351](https://github.com/charmbracelet/vhs/blob/f28239f3/THEMES.md?plain=1#L1-L351)

Theme Structure
---------------

A theme in VHS defines a comprehensive set of colors for the terminal. The `Theme` struct includes properties for background, foreground, cursor, and ANSI colors with their bright variants.

All color values are specified as hexadecimal strings (e.g., `"#171717"`). The default theme is defined in the `DefaultTheme` variable in `themes.go`.

Sources: [themes.go26-62](https://github.com/charmbracelet/vhs/blob/f28239f3/themes.go#L26-L62)
 [themes.go64-89](https://github.com/charmbracelet/vhs/blob/f28239f3/themes.go#L64-L89)

Customizing Themes
------------------

You can customize individual theme properties by passing a JSON object to the `Set Theme` command:

    Set Theme {"background": "#171717", "foreground": "#efefef"}
    

This will override specific properties while keeping the rest of the current theme. You can also apply these customizations on top of a named theme:

    Set Theme "Catppuccin Mocha"
    Set Theme {"cursor": "#ff00ff"}
    

### Theme Suggestion Mechanism

If you specify a theme name that doesn't exist, VHS will suggest similar theme names:

    invalid `Set Theme "catpuccin"`: did you mean "Catppuccin Mocha", "Catppuccin Latte"
    

VHS uses the Levenshtein distance algorithm to find similar theme names, and will suggest themes with a distance of 2 or less.

Terminal Styling Options
------------------------

Beyond color themes, VHS provides several options to enhance the visual appearance of your terminal recordings.

Sources: [draw.go177-210](https://github.com/charmbracelet/vhs/blob/f28239f3/draw.go#L177-L210)
 [draw.go212-228](https://github.com/charmbracelet/vhs/blob/f28239f3/draw.go#L212-L228)

### Window Bars

You can add a window bar (similar to macOS window decorations) to your terminal with:

    Set WindowBar <style>
    

Available styles are:

*   `Colorful` - macOS-style colored buttons on the left
*   `ColorfulRight` - Same as Colorful but on the right side
*   `Rings` - Ring-style buttons on the left
*   `RingsRight` - Ring-style buttons on the right

You can customize the window bar color and size:

    Set WindowBarSize 40
    Set WindowBarColor "#333333"
    

The window bar implementation is handled by the `MakeWindowBar` function, which calls either `makeColorfulBar` or `makeRingBar` based on the selected style.

Sources: [draw.go212-389](https://github.com/charmbracelet/vhs/blob/f28239f3/draw.go#L212-L389)

### Border Radius

You can add rounded corners to your terminal with:

    Set BorderRadius 10
    

The value specifies the radius of the corner in pixels. This is implemented using the `MakeBorderRadiusMask` function, which creates a PNG mask with rounded corners that is applied to the terminal.

Sources: [draw.go177-210](https://github.com/charmbracelet/vhs/blob/f28239f3/draw.go#L177-L210)
 [draw.go94-175](https://github.com/charmbracelet/vhs/blob/f28239f3/draw.go#L94-L175)

### Padding

You can add padding around your terminal:

    Set Padding 20
    

Or specify different padding for each side:

    Set PaddingLeft 10
    Set PaddingRight 10
    Set PaddingTop 5
    Set PaddingBottom 5
    

Complete Styling Example
------------------------

Here's an example tape file with comprehensive styling options:

    Output demo.gif
    Set FontSize 16
    Set Width 800
    Set Height 500
    
    # Apply a theme
    Set Theme "Catppuccin Mocha"
    
    # Customize specific colors
    Set Theme {"cursor": "#ff9900"}
    
    # Add styling elements
    Set Padding 20
    Set BorderRadius 10
    Set WindowBar Colorful
    Set WindowBarSize 30
    
    # Your terminal commands
    Type "echo 'This is a styled terminal example'"
    Enter
    Sleep 500ms
    

Theme Implementation Details
----------------------------

### Theme Loading Process

When VHS starts, it loads all predefined themes from the embedded `themes.json` file. The themes are parsed into `Theme` structs and stored in memory.

When you use the `Set Theme` command with a name, VHS:

1.  Searches for a theme with that exact name
2.  If found, applies the theme to the terminal
3.  If not found, searches for similar theme names to suggest
4.  Falls back to the default theme if no match is found

Technical Notes
---------------

*   The drawing functions in `draw.go` use Go's image packages to create PNG images for window bars and border radius masks
*   Theme colors are applied to the terminal through xterm.js configuration in the browser
*   VHS allows for fuzzy matching of theme names using the Levenshtein distance algorithm
*   The default theme is defined in the code and used as a fallback
