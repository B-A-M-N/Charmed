The Publishing feature in VHS enables you to share terminal GIFs by uploading them to charm.sh servers. This page documents how to publish recordings, the underlying SSH-based sharing infrastructure, and the various ways to share links with others. For information about running VHS as an SSH server, see [SSH Server](https://deepwiki.com/charmbracelet/vhs/4.2-ssh-server)
.

How to Publish
--------------

VHS offers three ways to publish a GIF:

1.  Using the dedicated `publish` command with an existing GIF:
    
        vhs publish demo.gif
        
    
2.  Using the `--publish` flag when running a tape:
    
        vhs --publish demo.tape
        
    
3.  Setting the `VHS_PUBLISH` environment variable to "true":
    
        VHS_PUBLISH=true vhs demo.tape

SSH Key Management
------------------

VHS automatically manages SSH keys for publishing:

*   Keys are stored in the user's data directory (determined by `dataPath()`)
*   If no key exists, a new Ed25519 key is generated using `keygen.New()`
*   Host keys are verified and stored in the known\_hosts file in the same directory
*   If a host key is new, it's automatically added to the known\_hosts file
*   If a host key has changed, VHS warns about potential man-in-the-middle attacks

Sharing Options
---------------

After publishing, VHS provides several ways to share the GIF:

1.  Markdown format:
    
        ![Made with VHS](https://vhs.charm.sh/your-unique-id.gif)
        
    
2.  HTML format (with badge):
    
3.  Direct URL:
    
        https://vhs.charm.sh/your-unique-id.gif
        
    

These instructions are displayed by the `publishShareInstructions()` function.

Technical Implementation
------------------------

### The Publish Function

The `Publish()` function handles the core publishing logic:

1.  Creates an SSH session with `sshSession()`
2.  Opens the GIF file
3.  Copies the file content to the SSH session's stdin
4.  Reads the URL from the session's stdout
5.  Returns the URL to the caller

    func Publish(ctx context.Context, path string) (string, error) {
        s, err := sshSession()
        // ...
        // Copy file to stdin
        _, err = io.Copy(in, f)
        // ...
        // Read URL from stdout
        b, err := io.ReadAll(out)
        // ...
        return string(b), nil
    }
    

### The SSH Session

The `sshSession()` function sets up the SSH connection:

1.  Gets the VHS data path with `dataPath()`
2.  Creates or loads an SSH key pair with `keygen.New()`
3.  Sets up the SSH client configuration with the key and host verification
4.  Connects to ghost.charm.sh:22
5.  Creates and returns a new SSH session

    func sshSession() (*ssh.Session, error) {
        dp, err := dataPath()
        // ...
        kp, err := keygen.New(filepath.Join(dp, "vhs_ed25519"), ...)
        // ...
        c, err := ssh.Dial("tcp", fmt.Sprintf("%s:%d", ghostHost, ghostPort), sshConfig)
        // ...
        s, err := c.NewSession()
        // ...
        return s, nil
    }
