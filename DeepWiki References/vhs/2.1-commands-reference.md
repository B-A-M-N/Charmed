This page provides a comprehensive reference for all commands available in VHS tape files. These commands control how terminal sessions are recorded and define the appearance and behavior of the output GIFs/videos. For information about output format options, see [Output Options](https://deepwiki.com/charmbracelet/vhs/2.2-output-options)
, and for styling and themes, see [Themes and Styling](https://deepwiki.com/charmbracelet/vhs/2.3-themes-and-styling)
.

Command Reference
-----------------

### Input Commands

#### Type

Types the specified text in the terminal.

**Syntax:** `Type[[@<speed>]] "<text>"`

**Arguments:**

*   `<text>`: The text to type
*   `@<speed>` (optional): Custom typing speed for this command

**Examples:**

    Type "Hello world"
    Type@100ms "Slow typing"
    

**Implementation:** [command.go327-356](https://github.com/charmbracelet/vhs/blob/f28239f3/command.go#L327-L356)

#### Key Commands

VHS provides commands for various special keys:

| Command | Description | Syntax | Example |
| --- | --- | --- | --- |
| Backspace | Press the backspace key | `Backspace [repeat] [@<speed>]` | `Backspace 3` |
| Delete | Press the delete key | `Delete [repeat] [@<speed>]` | `Delete` |
| Insert | Press the insert key | `Insert [repeat] [@<speed>]` | `Insert` |
| Down | Press the down arrow key | `Down [repeat] [@<speed>]` | `Down 2` |
| Enter | Press the enter key | `Enter [repeat] [@<speed>]` | `Enter` |
| Left | Press the left arrow key | `Left [repeat] [@<speed>]` | `Left 5` |
| Right | Press the right arrow key | `Right [repeat] [@<speed>]` | `Right 3` |
| Up  | Press the up arrow key | `Up [repeat] [@<speed>]` | `Up` |
| Tab | Press the tab key | `Tab [repeat] [@<speed>]` | `Tab 2` |
| Space | Press the space key | `Space [repeat] [@<speed>]` | `Space 4` |
| Escape | Press the escape key | `Escape [repeat] [@<speed>]` | `Escape` |
| PageUp | Press the page up key | `PageUp [repeat] [@<speed>]` | `PageUp` |
| PageDown | Press the page down key | `PageDown [repeat] [@<speed>]` | `PageDown` |

All key commands can take an optional repeat count to press the key multiple times, and an optional typing speed.

**Implementation:** [command.go78-104](https://github.com/charmbracelet/vhs/blob/f28239f3/command.go#L78-L104)

#### Modifier Commands

Commands for key combinations with modifier keys:

| Command | Description | Syntax | Example |
| --- | --- | --- | --- |
| Ctrl | Press key(s) with Ctrl held down | `Ctrl+<key(s)>` | `Ctrl+C`, `Ctrl+Alt+L` |
| Alt | Press key with Alt held down | `Alt+<key>` | `Alt+F4` |
| Shift | Press key with Shift held down | `Shift+<key>` | `Shift+Tab` |

**Implementation:** [command.go170-295](https://github.com/charmbracelet/vhs/blob/f28239f3/command.go#L170-L295)

### Control Commands

#### Hide and Show

Controls the visibility of terminal output in the recording.

| Command | Description | Example |
| --- | --- | --- |
| Hide | Pause recording | `Hide` |
| Show | Resume recording | `Show` |

**Implementation:** [command.go298-313](https://github.com/charmbracelet/vhs/blob/f28239f3/command.go#L298-L313)

#### Timing Commands

Commands that control timing in the recording:

| Command | Description | Syntax | Example |
| --- | --- | --- | --- |
| Sleep | Pause for specified duration | `Sleep <duration>` | `Sleep 1s`, `Sleep 500ms` |
| Wait | Wait for specific text to appear | `Wait[+Screen][@<timeout>] /<regexp>/` | `Wait Line /$ >/`, `Wait@5s Screen /Done/` |

The `Wait` command can look for text in the current line (`Line`) or entire screen (`Screen`), and supports regular expressions.

**Implementation:** [command.go318-325](https://github.com/charmbracelet/vhs/blob/f28239f3/command.go#L318-L325)
 [command.go110-169](https://github.com/charmbracelet/vhs/blob/f28239f3/command.go#L110-L169)

### Setup Commands

#### Output

Specifies the output file for the recording.

**Syntax:** `Output <path>.<format>`

**Arguments:**

*   `<path>`: The path to save the output file
*   `<format>`: File extension determining the output format (gif, mp4, webm)

**Examples:**

    Output demo.gif
    Output video/tutorial.mp4
    Output recordings/screencast.webm
    

**Implementation:** [command.go358-371](https://github.com/charmbracelet/vhs/blob/f28239f3/command.go#L358-L371)

#### Require

Ensures that a required command-line tool is available.

**Syntax:** `Require <program>`

**Examples:**

    Require ffmpeg
    Require git
    

**Implementation:** [command.go303-308](https://github.com/charmbracelet/vhs/blob/f28239f3/command.go#L303-L308)

#### Set

Configures various aspects of the recording.

**Syntax:** `Set <setting> <value>`

Available settings:

| Setting | Description | Type | Example |
| --- | --- | --- | --- |
| Shell | Shell to use | string | `Set Shell "bash"` |
| FontSize | Terminal font size | number | `Set FontSize 16` |
| FontFamily | Terminal font family | string | `Set FontFamily "Monaco"` |
| Height | Terminal height | number | `Set Height 600` |
| Width | Terminal width | number | `Set Width 800` |
| LetterSpacing | Space between letters | float | `Set LetterSpacing 1` |
| LineHeight | Height between lines | float | `Set LineHeight 1.2` |
| TypingSpeed | Default typing speed | duration | `Set TypingSpeed 50ms` |
| Theme | Terminal color theme | string or JSON | `Set Theme "Catppuccin Mocha"` |
| Padding | Padding around terminal | number | `Set Padding 20` |
| Framerate | Recording framerate | number | `Set Framerate 60` |
| PlaybackSpeed | Speed multiplier | float | `Set PlaybackSpeed 1.5` |
| WaitTimeout | Default timeout for Wait | duration | `Set WaitTimeout 5s` |
| WaitPattern | Default pattern for Wait | regexp | `Set WaitPattern "prompt$"` |
| LoopOffset | Loop point for GIFs | number or percentage | `Set LoopOffset 60.4` or `Set LoopOffset 20%` |
| MarginFill | Color of margin | string | `Set MarginFill "#1e1e1e"` |
| Margin | Size of margin | number | `Set Margin 20` |
| WindowBar | Type of window bar | string | `Set WindowBar "Rings"` |
| WindowBarSize | Size of window bar | number | `Set WindowBarSize 40` |
| BorderRadius | Corner radius | number | `Set BorderRadius 10` |
| CursorBlink | Whether cursor blinks | boolean | `Set CursorBlink false` |

**Implementation:** [command.go412-434](https://github.com/charmbracelet/vhs/blob/f28239f3/command.go#L412-L434)
 [command.go437-701](https://github.com/charmbracelet/vhs/blob/f28239f3/command.go#L437-L701)

Examples
--------

Here's a basic example tape file that demonstrates various commands:

    # Set output and configuration
    Output demo.gif
    Set FontSize 20
    Set Width 800
    Set Height 400
    Set TypingSpeed 100ms
    
    # Start recording
    Type "ls -la"
    Enter
    Sleep 2s
    
    # Hide some output
    Hide
    Type "clear"
    Enter
    Show
    
    # Demonstrate keyboard shortcuts
    Type "echo 'Hello VHS!'"
    Enter
    Sleep 1s
    Ctrl+L  # Clear screen
    Type "exit"
    Enter

Command Implementation Details
------------------------------

VHS commands are implemented as function handlers in the `CommandFuncs` map. Each command has a corresponding handler function that knows how to execute that specific command on a running VHS instance. When a command is executed, the appropriate handler is looked up and called with the parsed command and VHS instance.

The core execution function is `Execute()`, which takes a parsed command and VHS instance:

Each command handler then performs its specific action on the VHS instance, such as simulating keyboard input, modifying settings, or controlling the recording process.
