Understanding Tape File Structure
---------------------------------

A tape file is a simple text file containing a series of commands for VHS to execute. Here's a basic structure:

Let's examine a minimal example:

    # Specify the output file
    Output demo.gif
    
    # Configure the terminal
    Set FontSize 32
    Set Width 1200
    Set Height 600
    
    # Type a command
    Type "echo 'Welcome to VHS!'"
    Sleep 500ms
    Enter
    
    # Wait to see the output
    Sleep 5s
    

Sources: [examples/demo.tape1-70](https://github.com/charmbracelet/vhs/blob/f28239f3/examples/demo.tape#L1-L70)
 [README.md26-56](https://github.com/charmbracelet/vhs/blob/f28239f3/README.md?plain=1#L26-L56)

Essential Commands
------------------

Here are the most essential commands to get started:

| Command | Description | Example |
| --- | --- | --- |
| `Output` | Define output file format and location | `Output demo.gif` |
| `Set` | Configure terminal appearance | `Set FontSize 32` |
| `Type` | Type text into the terminal | `Type "echo Hello"` |
| `Enter` | Press Enter key | `Enter` |
| `Sleep` | Pause for specified duration | `Sleep 500ms` |
| `Hide`/`Show` | Toggle visibility of commands | `Hide`, `Show` |

Sources: [README.md218-236](https://github.com/charmbracelet/vhs/blob/f28239f3/README.md?plain=1#L218-L236)

Common Terminal Settings
------------------------

Here are some useful settings to customize your recording:

| Setting | Description | Example | Default |
| --- | --- | --- | --- |
| `FontSize` | Size of terminal font | `Set FontSize 32` | System default |
| `Width` | Width of terminal in pixels | `Set Width 1200` | System default |
| `Height` | Height of terminal in pixels | `Set Height 600` | System default |
| `TypingSpeed` | Delay between typed characters | `Set TypingSpeed 100ms` | 50ms |
| `Theme` | Terminal color theme | `Set Theme "Catppuccin Frappe"` | System default |
| `Padding` | Terminal padding in pixels | `Set Padding 20` | System default |

Sources: [README.md269-456](https://github.com/charmbracelet/vhs/blob/f28239f3/README.md?plain=1#L269-L456)

Troubleshooting
---------------

If you encounter problems:

1.  Ensure `ttyd` and `ffmpeg` are installed and on your `PATH`
2.  Check your tape file syntax with `vhs validate your.tape`
3.  Try hiding commands that might be causing issues with the `Hide` command

Sources: [README.md77-80](https://github.com/charmbracelet/vhs/blob/f28239f3/README.md?plain=1#L77-L80)

Example: Hello World
--------------------

Here's a complete example to try:

    Output hello.gif
    
    Set FontSize 32
    Set Width 800
    Set Height 400
    
    Type "echo 'Hello, VHS World!'"
    Sleep 500ms
    Enter
    Sleep 3s
    

Run with `vhs hello.tape` to get your first VHS recording.
