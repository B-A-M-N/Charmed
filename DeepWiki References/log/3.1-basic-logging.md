Global Logger Usage
-------------------

The `charmbracelet/log` package provides a global logger with sensible defaults that's immediately available for use. By default, the global logger:

*   Has logging level set to `InfoLevel`
*   Reports timestamps
*   Uses text formatter with colorized output

Log Levels
----------

The library provides five standard logging levels which control the verbosity and severity of log messages:

| Level | Function | Description | Default Visibility |
| --- | --- | --- | --- |
| `DebugLevel` | `log.Debug()` | Detailed information for debugging | Hidden by default |
| `InfoLevel` | `log.Info()` | Confirmation that things are working | Visible by default |
| `WarnLevel` | `log.Warn()` | Warning of potential issues | Visible by default |
| `ErrorLevel` | `log.Error()` | Error conditions that should be addressed | Visible by default |
| `FatalLevel` | `log.Fatal()` | Critical errors that terminate the program | Visible by default |

When you set a logging level, only messages at that level or higher will be displayed. For example, if you set the level to `WarnLevel`, then only warn, error, and fatal messages will be shown.
