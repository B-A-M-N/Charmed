Layout and Document Examples
============================

Relevant source files

*   [examples/layout/main.go](https://github.com/charmbracelet/lipgloss/blob/30441468/examples/layout/main.go)
    
*   [examples/list/sublist/main.go](https://github.com/charmbracelet/lipgloss/blob/30441468/examples/list/sublist/main.go)
    
*   [examples/ssh/main.go](https://github.com/charmbracelet/lipgloss/blob/30441468/examples/ssh/main.go)
    

This page documents the comprehensive layout example provided in the Lipgloss repository, which demonstrates how to compose complex terminal UIs using multiple layout primitives, adaptive colors, and compositing. The example showcases a complete terminal "document" with tabs, dialogs, color grids, lists, styled paragraphs, status bars, and layered content.

For basic styling patterns, see [Basic Styling Examples](https://deepwiki.com/charmbracelet/lipgloss/7.1-basic-styling-examples)
. For structured component usage, see [Component Examples](https://deepwiki.com/charmbracelet/lipgloss/7.3-component-examples)
. For advanced features like SSH integration and canvas operations, see [Advanced Integration Patterns](https://deepwiki.com/charmbracelet/lipgloss/7.4-advanced-integration-patterns)
.

Overview
--------

The layout example ([examples/layout/main.go1-434](https://github.com/charmbracelet/lipgloss/blob/30441468/examples/layout/main.go#L1-L434)
) creates a multi-section terminal UI that demonstrates real-world composition patterns. The document is built incrementally by composing individual sections and then combining them into a final output using layout functions.

### Document Structure

**Sources:** [examples/layout/main.go36-394](https://github.com/charmbracelet/lipgloss/blob/30441468/examples/layout/main.go#L36-L394)

The document is assembled in a linear fashion, with each section appended to a `strings.Builder` ([examples/layout/main.go218](https://github.com/charmbracelet/lipgloss/blob/30441468/examples/layout/main.go#L218-L218)
), before final rendering and compositing.

Component Breakdown
-------------------

### Tabs

The tab bar demonstrates horizontal joining with custom borders and alignment.

| Aspect | Implementation |
| --- | --- |
| Border Styles | `activeTabBorder` ([63-72](https://github.com/charmbracelet/lipgloss/blob/30441468/63-72)<br>) and `tabBorder` ([74-83](https://github.com/charmbracelet/lipgloss/blob/30441468/74-83)<br>) |
| Active Tab | `activeTab.Render("Lip Gloss")` ([224](https://github.com/charmbracelet/lipgloss/blob/30441468/224)<br>) |
| Inactive Tabs | `tab.Render()` for each tab ([225-228](https://github.com/charmbracelet/lipgloss/blob/30441468/225-228)<br>) |
| Gap Filling | `tabGap.Render()` with repeated spaces ([230](https://github.com/charmbracelet/lipgloss/blob/30441468/230)<br>) |
| Layout | `JoinHorizontal(lipgloss.Top, ...)` ([222](https://github.com/charmbracelet/lipgloss/blob/30441468/222)<br>) then `JoinHorizontal(lipgloss.Bottom, ...)` ([231](https://github.com/charmbracelet/lipgloss/blob/30441468/231)<br>) |

**Sources:** [examples/layout/main.go63-96](https://github.com/charmbracelet/lipgloss/blob/30441468/examples/layout/main.go#L63-L96)
 [examples/layout/main.go220-233](https://github.com/charmbracelet/lipgloss/blob/30441468/examples/layout/main.go#L220-L233)

The key technique is using different bottom border characters (`" "` for active, `"─"` for inactive) to create the visual effect that the active tab connects with the content below.

Key points:

*   Uses `uniseg.NewGraphemes()` to handle multi-byte characters correctly ([421](https://github.com/charmbracelet/lipgloss/blob/30441468/421)

### Color Grid Generation

The `colorGrid()` function creates smooth 2D color transitions:

Algorithm:

1.  Create left and right color columns by blending vertically ([398-399](https://github.com/charmbracelet/lipgloss/blob/30441468/398-399)
    )
2.  For each row, blend horizontally between left and right colors ([402-407](https://github.com/charmbracelet/lipgloss/blob/30441468/402-407)
    )
3.  Return 2D array of colors ([409](https://github.com/charmbracelet/lipgloss/blob/30441468/409)
    )

**Sources:** [examples/layout/main.go396-410](https://github.com/charmbracelet/lipgloss/blob/30441468/examples/layout/main.go#L396-L410)

### Color Downsampling

The final output uses `lipgloss.Println()` ([393](https://github.com/charmbracelet/lipgloss/blob/30441468/393)
), which automatically:

*   Detects terminal color capabilities
*   Downsamples colors to match terminal profile
*   Removes colors entirely for non-TTY output

This ensures the document renders correctly across different terminal types without manual adjustment.
