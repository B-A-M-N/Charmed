### Rendering Methods

The `Style` type provides multiple rendering approaches:

| Method | Usage Pattern | Output Destination |
| --- | --- | --- |
| `Render(str)` | `style.Render("text")` | Returns string |
| `SetString(str)` + `String()` | `style.SetString("text").String()` | Implements `Stringer` |
| Direct print | `lipgloss.Println(style)` | Terminal with color adaptation |

The `Render()` method accepts variadic string arguments and concatenates them before applying styling. The `SetString()` method stores text within the style for later rendering via the `String()` method.

### CSS-Style Shorthand

Both padding and margin support CSS-style shorthand notation:

| Arguments | Meaning | Example |
| --- | --- | --- |
| `(1)` | All sides | `Padding(2)` → 2 cells on all sides |
| `(1, 2)` | Vertical, Horizontal | `Margin(2, 4)` → 2 top/bottom, 4 left/right |
| `(1, 2, 3)` | Top, Horizontal, Bottom | `Padding(1, 4, 2)` → 1 top, 4 left/right, 2 bottom |
| `(1, 2, 3, 4)` | Top, Right, Bottom, Left | `Margin(2, 4, 3, 1)` → clockwise from top |
