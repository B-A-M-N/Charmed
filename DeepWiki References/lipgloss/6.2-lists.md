The Lists component provides a declarative API for rendering styled bullet lists and numbered lists in terminal applications. Lists support nesting, custom enumerators (bullets, numbers, roman numerals, etc.), and per-item styling.

For table rendering, see [Tables](https://deepwiki.com/charmbracelet/lipgloss/6.1-tables)
. For tree rendering, see [Trees](https://deepwiki.com/charmbracelet/lipgloss/6.3-trees)
. For general text layout, see [Layout and Composition](https://deepwiki.com/charmbracelet/lipgloss/5-layout-and-composition)
.

Overview
--------

The list package (`charm.land/lipgloss/v2/list`) is a higher-level component built on top of the core Lipgloss Style system. It handles enumeration, indentation, and styling for both simple and deeply nested list structures.

Basic List Creation
-------------------

Lists are created using `list.New()`, which accepts variadic string arguments for initial items. The list is rendered using the `String()` method or by printing with `lipgloss.Println()`.

List Structure and API
----------------------

The list package provides a fluent API for configuring list appearance and behavior. Key methods include:

| Method | Purpose | Example |
| --- | --- | --- |
| `list.New(items...)` | Create new list with initial items | `list.New("A", "B", "C")` |
| `.Item(value)` | Add an item to the list | `.Item("New Item")` |
| `.Enumerator(func)` | Set enumeration function | `.Enumerator(list.Roman)` |
| `.EnumeratorStyle(style)` | Set style for all enumerators | `.EnumeratorStyle(purple)` |
| `.EnumeratorStyleFunc(func)` | Set style function for enumerators | `.EnumeratorStyleFunc(fn)` |
| `.ItemStyle(style)` | Set style for all items | `.ItemStyle(pink)` |
| `.ItemStyleFunc(func)` | Set style function for items | `.ItemStyleFunc(fn)` |
| `.String()` | Render list to string | `l.String()` |

Enumerators
-----------

Enumerators determine how list items are marked. Lipgloss provides several built-in enumerators and supports custom enumerator functions.

### Built-in Enumerators

| Enumerator | Description | Example Output |
| --- | --- | --- |
| `list.Bullet` | Default bullet points | `•` |
| `list.Arabic` | Arabic numerals | `1.`, `2.`, `3.` |
| `list.Alphabet` | Alphabetic enumeration | `a.`, `b.`, `c.` |
| `list.Roman` | Roman numerals | `I.`, `II.`, `III.` |
| `list.Dash` | Dash markers | `-` |
| `list.Tree` | Tree-style markers | `├──`, `└──` |

### Custom Enumerator Functions

Enumerator functions have the signature: `func(list.Items, int) string`

They receive the full list of items and the current index, and return the enumerator string for that position.

Building Lists Incrementally
----------------------------

Lists can be constructed incrementally using the `.Item()` method, which is useful when the list contents are determined at runtime.

### Incremental Construction

From the README:

This pattern is useful for:

*   Building lists from dynamic data sources
*   Conditionally adding items
*   Creating lists in loops

### Items as Any Value Type

The `.Item()` method accepts any type. Strings are rendered directly, while nested lists (of type `list.List`) create hierarchical structures. Other types are converted to strings during rendering.

**Sources:** [README.md688-696](https://github.com/charmbracelet/lipgloss/blob/30441468/README.md?plain=1#L688-L696)

Summary
-------

The List component provides:

1.  **Simple API**: `list.New()` with fluent method chaining
2.  **Flexible Enumeration**: Built-in enumerators (bullets, numbers, roman numerals) and custom functions
3.  **Nested Structure**: Arbitrary nesting depth with independent styling
4.  **Rich Styling**: Per-item and per-enumerator styling with functions or uniform styles
5.  **Terminal Adaptation**: Automatic integration with Lipgloss color downsampling

Lists are ideal for presenting hierarchical data, checklists, menus, and structured content in terminal applications.
