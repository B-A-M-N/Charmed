Purpose and Scope
-----------------

This page documents the color type system in Lipgloss, including color definitions, parsing from multiple input formats, and color manipulation utilities. The color system provides a flexible API for specifying colors in various formats (hex, ANSI, ANSI256, RGB integers) and transforming them through manipulation functions.

For adaptive color selection based on terminal capabilities and background detection, see [Adaptive Colors and Background Detection](https://deepwiki.com/charmbracelet/lipgloss/3.2-adaptive-colors-and-background-detection)
. For automatic color downsampling and output writers, see [Terminal Adaptation and Output](https://deepwiki.com/charmbracelet/lipgloss/3.3-terminal-adaptation-and-output)
.

Color Type Architecture
-----------------------

The color system is built on Go's standard `color.Color` interface and provides several concrete implementations for different use cases.

**Sources:** [color.go23-162](https://github.com/charmbracelet/lipgloss/blob/30441468/color.go#L23-L162)

### NoColor Type

`NoColor` represents the absence of color styling. When used as a foreground color, text renders with the terminal's default text color. When used as a background color, no background is drawn.

| Property | Value |
| --- | --- |
| Definition | [color.go45-52](https://github.com/charmbracelet/lipgloss/blob/30441468/color.go#L45-L52) |
| RGBA Returns | `(0x0, 0x0, 0x0, 0xFFFF)` |
| Use Case | Explicitly remove color styling from a Style |

**Sources:** [color.go43-61](https://github.com/charmbracelet/lipgloss/blob/30441468/color.go#L43-L61)

### ANSI Color Constants

The system provides 16 pre-defined 4-bit ANSI color constants:

| Constant | Value | Range |
| --- | --- | --- |
| `Black` through `White` | 0-7 | Standard colors |
| `BrightBlack` through `BrightWhite` | 8-15 | Bright variants |

These constants are of type `ansi.BasicColor` and can be used directly in style setters.

**Sources:** [color.go23-41](https://github.com/charmbracelet/lipgloss/blob/30441468/color.go#L23-L41)

### RGBColor Type

`RGBColor` is a struct representing a color by its red, green, and blue components:

The `RGBA()` method [color.go146-153](https://github.com/charmbracelet/lipgloss/blob/30441468/color.go#L146-L153)
 converts the 8-bit components to 32-bit values by shifting and returns full opacity (0xFFFF).

**Sources:** [color.go137-153](https://github.com/charmbracelet/lipgloss/blob/30441468/color.go#L137-L153)

### ANSIColor Type

`ANSIColor` is an alias for `ansi.IndexedColor` [color.go161](https://github.com/charmbracelet/lipgloss/blob/30441468/color.go#L161-L161)
 and represents colors in the ANSI256 color palette (values 16-255).

**Sources:** [color.go155-161](https://github.com/charmbracelet/lipgloss/blob/30441468/color.go#L155-L161)

Color Parsing
-------------

The `Color(string)` function [color.go68-95](https://github.com/charmbracelet/lipgloss/blob/30441468/color.go#L68-L95)
 is the primary interface for creating colors from string representations. It accepts multiple formats and returns a `color.Color`.

**Supported Formats:**

| Format | Example | Parsed As | Code Range |
| --- | --- | --- | --- |
| Hex 6-digit | `"#FF0000"` | `color.RGBA` via `parseHex()` | [color.go69-74](https://github.com/charmbracelet/lipgloss/blob/30441468/color.go#L69-L74) |
| Hex 3-digit | `"#F00"` | `color.RGBA` via `parseHex()` | [color.go69-74](https://github.com/charmbracelet/lipgloss/blob/30441468/color.go#L69-L74) |
| ANSI (0-15) | `"1"` | `ansi.BasicColor` | [color.go87-88](https://github.com/charmbracelet/lipgloss/blob/30441468/color.go#L87-L88) |
| ANSI256 (16-255) | `"21"` | `ANSIColor` | [color.go89-90](https://github.com/charmbracelet/lipgloss/blob/30441468/color.go#L89-L90) |
| RGB Integer | `"16711680"` | `color.RGBA` | [color.go93-94](https://github.com/charmbracelet/lipgloss/blob/30441468/color.go#L93-L94) |
| RGB Hex Integer | (internal) | `color.RGBA` | [color.go93-94](https://github.com/charmbracelet/lipgloss/blob/30441468/color.go#L93-L94) |

**Sources:** [color.go63-95](https://github.com/charmbracelet/lipgloss/blob/30441468/color.go#L63-L95)

### Hex Color Parsing

The `parseHex()` function [color.go102-135](https://github.com/charmbracelet/lipgloss/blob/30441468/color.go#L102-L135)
 provides optimized parsing of hex color strings without external dependencies. It handles both 6-digit (#RRGGBB) and 3-digit (#RGB) formats.

**Parsing Logic:**

*   **6-digit format** [color.go124-126](https://github.com/charmbracelet/lipgloss/blob/30441468/color.go#L124-L126)
    : Each pair of hex digits is converted to a byte (e.g., "FF" → 255)
*   **3-digit format** [color.go128-130](https://github.com/charmbracelet/lipgloss/blob/30441468/color.go#L128-L130)
    : Each hex digit is multiplied by 17 to expand (e.g., "F" → "FF" → 255)
*   **Case insensitive**: Accepts both uppercase and lowercase hex digits [color.go109-120](https://github.com/charmbracelet/lipgloss/blob/30441468/color.go#L109-L120)
    

**Sources:** [color.go99-135](https://github.com/charmbracelet/lipgloss/blob/30441468/color.go#L99-L135)
 [color\_test.go119-174](https://github.com/charmbracelet/lipgloss/blob/30441468/color_test.go#L119-L174)

### RGB Integer Parsing

For numeric strings that represent values ≥ 256, the `Color()` function interprets them as RGB integers [color.go93-94](https://github.com/charmbracelet/lipgloss/blob/30441468/color.go#L93-L94)
 The integer is unpacked using bit operations:

*   Red: `(i >> 16) & 0xFF`
*   Green: `(i >> 8) & 0xFF`
*   Blue: `i & 0xFF`

**Sources:** [color.go93-94](https://github.com/charmbracelet/lipgloss/blob/30441468/color.go#L93-L94)

Color Manipulation Functions
----------------------------

Lipgloss provides four core color manipulation functions that transform colors while preserving the original color object (returning new colors).

**Sources:** [color.go290-359](https://github.com/charmbracelet/lipgloss/blob/30441468/color.go#L290-L359)

### Alpha Adjustment

The `Alpha(c color.Color, alpha float64)` function [color.go292-304](https://github.com/charmbracelet/lipgloss/blob/30441468/color.go#L292-L304)
 adjusts the opacity of a color.

**Parameters:**

*   `c`: Input color (returns `nil` if input is `nil`)
*   `alpha`: Opacity value (0.0 = transparent, 1.0 = opaque), clamped to \[0, 1\]

**Implementation:**

1.  Extracts RGB components from input color [color.go297](https://github.com/charmbracelet/lipgloss/blob/30441468/color.go#L297-L297)
    
2.  Converts 32-bit values to 8-bit (divide by 256) [color.go299-301](https://github.com/charmbracelet/lipgloss/blob/30441468/color.go#L299-L301)
    
3.  Applies clamped alpha value scaled to \[0, 255\] [color.go302](https://github.com/charmbracelet/lipgloss/blob/30441468/color.go#L302-L302)
    
4.  Returns new `color.RGBA`

**Example Use Cases:**

*   Creating semi-transparent overlays
*   Fading text or backgrounds
*   Gradient effects with varying opacity

**Sources:** [color.go290-304](https://github.com/charmbracelet/lipgloss/blob/30441468/color.go#L290-L304)
 [color\_test.go176-233](https://github.com/charmbracelet/lipgloss/blob/30441468/color_test.go#L176-L233)

### Complementary Colors

The `Complementary(c color.Color)` function [color.go308-325](https://github.com/charmbracelet/lipgloss/blob/30441468/color.go#L308-L325)
 returns the complementary color (opposite on the color wheel).

**Algorithm:**

1.  Converts input to `colorful.Color` using `go-colorful` library [color.go314](https://github.com/charmbracelet/lipgloss/blob/30441468/color.go#L314-L314)
    
2.  Extracts HSV (Hue, Saturation, Value) components [color.go316](https://github.com/charmbracelet/lipgloss/blob/30441468/color.go#L316-L316)
    
3.  Rotates hue by 180° [color.go317-322](https://github.com/charmbracelet/lipgloss/blob/30441468/color.go#L317-L322)
    :
    *   Adds 180° to hue
    *   Normalizes to [0, 360) range\\n4. Converts back to RGB and clamps \[color.go324](https://github.com/charmbracelet/lipgloss/blob/30441468/0,%20360)%20range/n4.%20Converts%20back%20to%20RGB%20and%20clamps%20[color.go#L324-L324)\
        \
\
**Special Cases:**\
\
*   Black, white, and gray (zero saturation) return themselves (no hue to complement) [color\_test.go248-252](https://github.com/charmbracelet/lipgloss/blob/30441468/color_test.go#L248-L252)\
    \
*   `nil` input returns `nil` [color.go309-311](https://github.com/charmbracelet/lipgloss/blob/30441468/color.go#L309-L311)\
    \
\
**Sources:** [color.go306-325](https://github.com/charmbracelet/lipgloss/blob/30441468/color.go#L306-L325)\
 [color\_test.go235-264](https://github.com/charmbracelet/lipgloss/blob/30441468/color_test.go#L235-L264)\
\
### Darken Function\
\
The `Darken(c color.Color, percent float64)` function [color.go328-342](https://github.com/charmbracelet/lipgloss/blob/30441468/color.go#L328-L342)\
 reduces the brightness of a color.\
\
**Parameters:**\
\
*   `c`: Input color\
*   `percent`: Darkening percentage (0.0-1.0, clamped)\
\
**Implementation:**\
\
1.  Calculates multiplier: `mult = 1.0 - percent` [color.go333](https://github.com/charmbracelet/lipgloss/blob/30441468/color.go#L333-L333)\
    \
2.  Multiplies each RGB component by multiplier [color.go337-339](https://github.com/charmbracelet/lipgloss/blob/30441468/color.go#L337-L339)\
    \
3.  Preserves alpha channel [color.go340](https://github.com/charmbracelet/lipgloss/blob/30441468/color.go#L340-L340)\
    \
\
**Behavior:**\
\
*   `percent = 0.0`: No change (multiplier = 1.0)\
*   `percent = 0.5`: 50% darker (multiplier = 0.5)\
*   `percent = 1.0`: Pure black (multiplier = 0.0)\
\
**Sources:** [color.go327-342](https://github.com/charmbracelet/lipgloss/blob/30441468/color.go#L327-L342)\
 [color\_test.go266-288](https://github.com/charmbracelet/lipgloss/blob/30441468/color_test.go#L266-L288)\
\
### Lighten Function\
\
The `Lighten(c color.Color, percent float64)` function [color.go345-359](https://github.com/charmbracelet/lipgloss/blob/30441468/color.go#L345-L359)\
 increases the brightness of a color.\
\
**Parameters:**\
\
*   `c`: Input color\
*   `percent`: Lightening percentage (0.0-1.0, clamped)\
\
**Implementation:**\
\
1.  Calculates additive value: `add = 255 * percent` [color.go350](https://github.com/charmbracelet/lipgloss/blob/30441468/color.go#L350-L350)\
    \
2.  Adds to each RGB component [color.go354-356](https://github.com/charmbracelet/lipgloss/blob/30441468/color.go#L354-L356)\
    \
3.  Clamps results to \[0, 255\] using `min(255, ...)` [color.go354-356](https://github.com/charmbracelet/lipgloss/blob/30441468/color.go#L354-L356)\
    \
4.  Preserves alpha channel [color.go357](https://github.com/charmbracelet/lipgloss/blob/30441468/color.go#L357-L357)\
    \
\
**Behavior:**\
\
*   `percent = 0.0`: No change (add = 0)\
*   `percent = 0.5`: Halfway to white (add = 127.5)\
*   `percent = 1.0`: Pure white (add = 255)\
\
**Sources:** [color.go344-359](https://github.com/charmbracelet/lipgloss/blob/30441468/color.go#L344-L359)\
 [color\_test.go290-312](https://github.com/charmbracelet/lipgloss/blob/30441468/color_test.go#L290-L312)\
\
Color Manipulation Comparison\
-----------------------------\
\
| Function | Method | Range | Preserves |\
| --- | --- | --- | --- |\
| `Alpha` | Scales alpha channel | \[0, 1\] → \[0, 255\] | RGB values |\
| `Complementary` | Rotates hue 180° in HSV | Hue: \[0, 360)° | Saturation, Value |\
| `Darken` | Multiplies RGB by (1-percent) | \[0, 1\] → multiplier | Alpha |\
| `Lighten` | Adds (255\*percent) to RGB | \[0, 1\] → additive | Alpha |\
\
**Sources:** [color.go290-359](https://github.com/charmbracelet/lipgloss/blob/30441468/color.go#L290-L359)\

### ensureNotTransparent Function\
\
The `ensureNotTransparent(c color.Color)` helper [color.go282-288](https://github.com/charmbracelet/lipgloss/blob/30441468/color.go#L282-L288)\
 ensures colors have at least 1/255 opacity. This is used for gradient purposes when colors might lose their alpha channel during conversion.\
\
**Logic:**\
\
1.  Checks if alpha component is 0 [color.go284](https://github.com/charmbracelet/lipgloss/blob/30441468/color.go#L284-L284)\
    \
2.  If zero, applies `Alpha(c, 1)` to set minimum opacity [color.go285](https://github.com/charmbracelet/lipgloss/blob/30441468/color.go#L285-L285)\
    \
3.  Otherwise returns color unchanged [color.go287](https://github.com/charmbracelet/lipgloss/blob/30441468/color.go#L287-L287)\
    \
\
**Sources:** [color.go279-288](https://github.com/charmbracelet/lipgloss/blob/30441468/color.go#L279-L288)\

Integration with External Libraries\
-----------------------------------\
\

\
**Dependencies:**\
\
| Library | Purpose | Usage in Color System |\
| --- | --- | --- |\
