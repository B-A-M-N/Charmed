Style Configuration
===================

Relevant source files

*   [borders.go](https://github.com/charmbracelet/lipgloss/blob/30441468/borders.go)
    
*   [borders\_test.go](https://github.com/charmbracelet/lipgloss/blob/30441468/borders_test.go)
    
*   [get.go](https://github.com/charmbracelet/lipgloss/blob/30441468/get.go)
    
*   [set.go](https://github.com/charmbracelet/lipgloss/blob/30441468/set.go)
    
*   [style.go](https://github.com/charmbracelet/lipgloss/blob/30441468/style.go)
    
*   [unset.go](https://github.com/charmbracelet/lipgloss/blob/30441468/unset.go)
    

This page documents how to configure style properties in lipgloss using the fluent API, including the internal architecture of property storage through the `props` bitmask and individual style fields. It covers setting, getting, unsetting, and inheriting style properties.

For information about the rendering pipeline that applies these configured properties, see [Core Style System](https://deepwiki.com/charmbracelet/lipgloss/2-core-style-system)
. For details about specific property types like colors and borders, see [Colors](https://deepwiki.com/charmbracelet/lipgloss/2.2-text-attributes-and-styling)
 and [Borders](https://deepwiki.com/charmbracelet/lipgloss/2.3-rendering-pipeline)
.

* * *

Property Storage Architecture
-----------------------------

Lipgloss uses a dual-storage system for tracking style properties: a bitmask to track **which** properties are set, and individual struct fields to store the property **values**.

### The Props Bitmask System

**Sources:** [style.go14-79](https://github.com/charmbracelet/lipgloss/blob/30441468/style.go#L14-L79)
 [style.go82-97](https://github.com/charmbracelet/lipgloss/blob/30441468/style.go#L82-L97)
 [style.go116-161](https://github.com/charmbracelet/lipgloss/blob/30441468/style.go#L116-L161)

The `propKey` type represents individual properties as bit-shifted constants [style.go14-79](https://github.com/charmbracelet/lipgloss/blob/30441468/style.go#L14-L79)
 Each property occupies a single bit position in a 64-bit integer. The `props` type [style.go82](https://github.com/charmbracelet/lipgloss/blob/30441468/style.go#L82-L82)
 is an int64 bitmask that tracks which properties have been explicitly set on a style.

| Property Category | propKey Constants | Bit Range |
| --- | --- | --- |
| Boolean Attributes | `boldKey` through `colorWhitespaceKey` | Bits 0-9 |
| Colors & Dimensions | `foregroundKey` through `alignVerticalKey` | Bits 10-16 |
| Padding | `paddingTopKey` through `paddingLeftKey` | Bits 17-20 |
| Margins | `marginTopKey` through `marginBackgroundKey` | Bits 21-25 |
| Borders | `borderStyleKey` through `borderLeftBackgroundKey` | Bits 26-37 |
| Advanced | `inlineKey` through `transformKey` | Bits 38-42 |

The `props` type provides three fundamental operations [style.go84-97](https://github.com/charmbracelet/lipgloss/blob/30441468/style.go#L84-L97)
:

*   **`set(k propKey)`**: Sets the bit for property `k` using bitwise OR
*   **`unset(k propKey)`**: Clears the bit for property `k` using bitwise AND-NOT
*   **`has(k propKey)`**: Checks if property `k` is set using bitwise AND

### The Style Struct

The `Style` struct stores both the tracking bitmask and the actual property values:

**Sources:** [style.go116-161](https://github.com/charmbracelet/lipgloss/blob/30441468/style.go#L116-L161)

Boolean attributes (`bold`, `italic`, `underline`, etc.) are stored compactly in a single `attrs` integer field [style.go123](https://github.com/charmbracelet/lipgloss/blob/30441468/style.go#L123-L123)
 while non-boolean properties have dedicated fields [style.go126-160](https://github.com/charmbracelet/lipgloss/blob/30441468/style.go#L126-L160)
 This separation optimizes memory usage while maintaining type safety for complex properties.

Setting Properties
------------------

Properties are configured using fluent setter methods that return a modified `Style`, enabling method chaining. All setters follow the pattern: accept a value, update the internal storage, and return the style.

### Internal Set Mechanism

**Sources:** [set.go3-87](https://github.com/charmbracelet/lipgloss/blob/30441468/set.go#L3-L87)

The internal `set()` method [set.go4-87](https://github.com/charmbracelet/lipgloss/blob/30441468/set.go#L4-L87)
 handles all property updates. It performs type-specific storage (e.g., ensuring non-negative integers for dimensions [set.go14-15](https://github.com/charmbracelet/lipgloss/blob/30441468/set.go#L14-L15)
), then marks the property as set in the bitmask [set.go86](https://github.com/charmbracelet/lipgloss/blob/30441468/set.go#L86-L86)

### Text Attribute Setters

Text formatting properties control font rendering characteristics:

| Method | Property | Description |
| --- | --- | --- |
| `Bold(bool)` | `boldKey` | Bold text weight |
| `Italic(bool)` | `italicKey` | Italic text style |
| `Underline(bool)` | `underlineKey` | Underline text |
| `Strikethrough(bool)` | `strikethroughKey` | Strike through text |
| `Reverse(bool)` | `reverseKey` | Invert foreground/background |
| `Blink(bool)` | `blinkKey` | Blinking text |
| `Faint(bool)` | `faintKey` | Dimmed text |

**Sources:** [set.go161-206](https://github.com/charmbracelet/lipgloss/blob/30441468/set.go#L161-L206)

Each method updates the corresponding bit in the `attrs` field [set.go69-82](https://github.com/charmbracelet/lipgloss/blob/30441468/set.go#L69-L82)
 based on the boolean value passed.

### Color Setters

**Sources:** [set.go208-224](https://github.com/charmbracelet/lipgloss/blob/30441468/set.go#L208-L224)
 [set.go385-388](https://github.com/charmbracelet/lipgloss/blob/30441468/set.go#L385-L388)

Color setters accept any type implementing `TerminalColor` [set.go215-217](https://github.com/charmbracelet/lipgloss/blob/30441468/set.go#L215-L217)
 which is validated through `colorOrNil()` [set.go154-159](https://github.com/charmbracelet/lipgloss/blob/30441468/set.go#L154-L159)
 To remove a color, pass `lipgloss.NoColor`.

### Dimension Setters

Dimension setters control the size constraints of rendered blocks:

| Method | Property | Behavior | Validation |
| --- | --- | --- | --- |
| `Width(int)` | `widthKey` | Sets block width, enables text wrapping | `max(0, value)` |
| `Height(int)` | `heightKey` | Sets block height with vertical alignment | `max(0, value)` |
| `MaxWidth(int)` | `maxWidthKey` | Truncates lines exceeding this width | `max(0, value)` |
| `MaxHeight(int)` | `maxHeightKey` | Truncates content exceeding this height | `max(0, value)` |

**Sources:** [set.go226-240](https://github.com/charmbracelet/lipgloss/blob/30441468/set.go#L226-L240)
 [set.go612-641](https://github.com/charmbracelet/lipgloss/blob/30441468/set.go#L612-L641)

All dimension values are clamped to non-negative integers during storage [set.go15-17](https://github.com/charmbracelet/lipgloss/blob/30441468/set.go#L15-L17)

### Padding and Margin Setters

Spacing setters follow CSS-style shorthand syntax:

**Sources:** [set.go269-294](https://github.com/charmbracelet/lipgloss/blob/30441468/set.go#L269-L294)
 [set.go331-356](https://github.com/charmbracelet/lipgloss/blob/30441468/set.go#L331-L356)
 [set.go695-733](https://github.com/charmbracelet/lipgloss/blob/30441468/set.go#L695-L733)

The `whichSidesInt()` helper [set.go705-733](https://github.com/charmbracelet/lipgloss/blob/30441468/set.go#L705-L733)
 implements CSS shorthand rules. Individual side methods (`PaddingLeft()`, `MarginTop()`, etc.) are also available [set.go296-318](https://github.com/charmbracelet/lipgloss/blob/30441468/set.go#L296-L318)
 [set.go358-380](https://github.com/charmbracelet/lipgloss/blob/30441468/set.go#L358-L380)

### Border Setters

Border configuration uses a two-step process: defining the border style, then enabling specific sides:

**Sources:** [set.go390-449](https://github.com/charmbracelet/lipgloss/blob/30441468/set.go#L390-L449)

The `Border()` method [set.go413-430](https://github.com/charmbracelet/lipgloss/blob/30441468/set.go#L413-L430)
 accepts a `Border` struct and optional boolean arguments using the same CSS shorthand pattern. If no sides are specified, all four sides default to enabled [set.go417-422](https://github.com/charmbracelet/lipgloss/blob/30441468/set.go#L417-L422)

Border colors can be set per-side for both foreground and background:

| Method Pattern | Shorthand | Individual |
| --- | --- | --- |
| Foreground | `BorderForeground(...TerminalColor)` | `BorderTopForeground()`, `BorderRightForeground()`, etc. |
| Background | `BorderBackground(...TerminalColor)` | `BorderTopBackground()`, `BorderRightBackground()`, etc. |

**Sources:** [set.go475-506](https://github.com/charmbracelet/lipgloss/blob/30441468/set.go#L475-L506)
 [set.go508-592](https://github.com/charmbracelet/lipgloss/blob/30441468/set.go#L508-L592)

### Alignment Setters

Alignment uses the `Position` type, a float64 representing relative position (0.0 = left/top, 1.0 = right/bottom):

**Sources:** [set.go241-267](https://github.com/charmbracelet/lipgloss/blob/30441468/set.go#L241-L267)

Individual alignment methods are also available: `AlignHorizontal()` and `AlignVertical()` [set.go257-267](https://github.com/charmbracelet/lipgloss/blob/30441468/set.go#L257-L267)

### Special Setters

| Method | Property | Description |
| --- | --- | --- |
| `Inline(bool)` | `inlineKey` | Disables padding/margin/borders, renders as single line |
| `TabWidth(int)` | `tabWidthKey` | Spaces per tab character (default: 4, -1 disables conversion) |
| `UnderlineSpaces(bool)` | `underlineSpacesKey` | Whether underlines apply to spaces |
| `StrikethroughSpaces(bool)` | `strikethroughSpacesKey` | Whether strikethroughs apply to spaces |
| `Transform(func(string) string)` | `transformKey` | Custom string transformation function |

**Sources:** [set.go594-686](https://github.com/charmbracelet/lipgloss/blob/30441468/set.go#L594-L686)

The `Transform()` method stores a function pointer that will be applied to text during rendering [set.go683-686](https://github.com/charmbracelet/lipgloss/blob/30441468/set.go#L683-L686)

Getting Property Values
-----------------------

Getter methods retrieve configured property values, returning sensible defaults for unset properties.

### Internal Get Mechanism

**Sources:** [get.go417-465](https://github.com/charmbracelet/lipgloss/blob/30441468/get.go#L417-L465)
 [get.go467-541](https://github.com/charmbracelet/lipgloss/blob/30441468/get.go#L467-L541)

All getters first check `isSet()` [get.go418-420](https://github.com/charmbracelet/lipgloss/blob/30441468/get.go#L418-L420)
 to determine if the property has been configured. If not set, they return a default value appropriate to the property type.

### Boolean Attribute Getters

**Sources:** [get.go9-48](https://github.com/charmbracelet/lipgloss/blob/30441468/get.go#L9-L48)
 [get.go422-427](https://github.com/charmbracelet/lipgloss/blob/30441468/get.go#L422-L427)

Boolean getters extract the relevant bit from the `attrs` field [get.go426](https://github.com/charmbracelet/lipgloss/blob/30441468/get.go#L426-L426)
 All boolean properties default to `false` when unset.

### Color Getters

| Method | Property | Default |
| --- | --- | --- |
| `GetForeground()` | `foregroundKey` | `noColor` |
| `GetBackground()` | `backgroundKey` | `noColor` |
| `GetBorderTopForeground()` | `borderTopForegroundKey` | `noColor` |
| `GetMarginBackground()` | `marginBackgroundKey` | `noColor` |

**Sources:** [get.go50-60](https://github.com/charmbracelet/lipgloss/blob/30441468/get.go#L50-L60)
 [get.go242-288](https://github.com/charmbracelet/lipgloss/blob/30441468/get.go#L242-L288)

Color getters return `noColor` (an internal sentinel value) when the property is unset [get.go429-465](https://github.com/charmbracelet/lipgloss/blob/30441468/get.go#L429-L465)

### Dimension and Spacing Getters

Dimension getters return `0` for unset values [get.go64-72](https://github.com/charmbracelet/lipgloss/blob/30441468/get.go#L64-L72)
:

**Sources:** [get.go64-198](https://github.com/charmbracelet/lipgloss/blob/30441468/get.go#L64-L198)
 [get.go389-409](https://github.com/charmbracelet/lipgloss/blob/30441468/get.go#L389-L409)

Compound getters like `GetPadding()` [get.go106-111](https://github.com/charmbracelet/lipgloss/blob/30441468/get.go#L106-L111)
 and `GetHorizontalFrameSize()` [get.go393-395](https://github.com/charmbracelet/lipgloss/blob/30441468/get.go#L393-L395)
 aggregate multiple dimension values.

### Border Getters

Border getters combine style information with visibility flags:

| Method | Returns | Notes |
| --- | --- | --- |
| `GetBorder()` | `(Border, top, right, bottom, left bool)` | Complete border configuration |
| `GetBorderStyle()` | `Border` | Border character definitions |
| `GetBorderTop()` | `bool` | Top border visibility |
| `GetBorderTopSize()` | `int` | Width of top border runes |

**Sources:** [get.go200-351](https://github.com/charmbracelet/lipgloss/blob/30441468/get.go#L200-L351)

Size getters account for implicit borders [get.go302-307](https://github.com/charmbracelet/lipgloss/blob/30441468/get.go#L302-L307)
 where a border style is set but no sides are explicitly enabled (defaults to all sides).

Unsetting Properties
--------------------

Unset methods remove properties from a style, reverting them to default behavior.

### Unset Mechanism

**Sources:** [unset.go3-6](https://github.com/charmbracelet/lipgloss/blob/30441468/unset.go#L3-L6)

The internal `unset()` method [unset.go4-6](https://github.com/charmbracelet/lipgloss/blob/30441468/unset.go#L4-L6)
 clears the property's bit in the `props` bitmask using bitwise AND-NOT. The actual field value remains in memory but is ignored during rendering and getter operations.

### Unset Methods by Category

All properties have corresponding unset methods following the pattern `Unset<PropertyName>()`:

| Category | Example Methods |
| --- | --- |
| Text Attributes | `UnsetBold()`, `UnsetItalic()`, `UnsetUnderline()` |
| Colors | `UnsetForeground()`, `UnsetBackground()` |
| Dimensions | `UnsetWidth()`, `UnsetHeight()`, `UnsetMaxWidth()` |
| Spacing | `UnsetPadding()`, `UnsetPaddingLeft()`, `UnsetMargins()` |
| Borders | `UnsetBorderStyle()`, `UnsetBorderTop()`, `UnsetBorderForeground()` |
| Special | `UnsetInline()`, `UnsetTransform()`, `UnsetString()` |

**Sources:** [unset.go8-331](https://github.com/charmbracelet/lipgloss/blob/30441468/unset.go#L8-L331)

Compound unset methods like `UnsetPadding()` [unset.go93-100](https://github.com/charmbracelet/lipgloss/blob/30441468/unset.go#L93-L100)
 and `UnsetBorderBackground()` [unset.go242-248](https://github.com/charmbracelet/lipgloss/blob/30441468/unset.go#L242-L248)
 remove multiple related properties in a single call.

Style Inheritance
-----------------

The `Inherit()` method overlays one style onto another, copying unset properties from the source style.

### Inheritance Algorithm

**Sources:** [style.go199-231](https://github.com/charmbracelet/lipgloss/blob/30441468/style.go#L199-L231)

The inheritance algorithm [style.go204-230](https://github.com/charmbracelet/lipgloss/blob/30441468/style.go#L204-L230)
:

1.  **Iterates through all property keys** from `boldKey` to `transformKey` [style.go205](https://github.com/charmbracelet/lipgloss/blob/30441468/style.go#L205-L205)
    
2.  **Skips unset properties** in the source style [style.go206-208](https://github.com/charmbracelet/lipgloss/blob/30441468/style.go#L206-L208)
    
3.  **Excludes margins and padding** which are never inherited [style.go210-216](https://github.com/charmbracelet/lipgloss/blob/30441468/style.go#L210-L216)
    
4.  **Special case for background color**: If neither style has an explicit margin background, but the source has a background color, the margin background inherits that color [style.go217-222](https://github.com/charmbracelet/lipgloss/blob/30441468/style.go#L217-L222)
    
5.  **Skips already-set properties** in the target style [style.go224-226](https://github.com/charmbracelet/lipgloss/blob/30441468/style.go#L224-L226)
    
6.  **Copies the value** using `setFrom()` [style.go228](https://github.com/charmbracelet/lipgloss/blob/30441468/style.go#L228-L228)
    

### The setFrom Method

**Sources:** [set.go89-152](https://github.com/charmbracelet/lipgloss/blob/30441468/set.go#L89-L152)

The `setFrom()` method [set.go90-152](https://github.com/charmbracelet/lipgloss/blob/30441468/set.go#L90-L152)
 extracts the appropriate field from the source style based on the property key [set.go92-147](https://github.com/charmbracelet/lipgloss/blob/30441468/set.go#L92-L147)
 then calls the standard `set()` method to store it [set.go93](https://github.com/charmbracelet/lipgloss/blob/30441468/set.go#L93-L93)

### Inheritance Example Properties

| Property Category | Inherited? | Notes |
| --- | --- | --- |
| Bold, Italic, Underline | ✓ Yes | All text attributes inherit |
| Foreground, Background | ✓ Yes | Colors inherit with terminal awareness |
| Width, Height | ✓ Yes | Dimensions inherit |
| Padding (all sides) | ✗ No | Explicitly excluded [style.go214-216](https://github.com/charmbracelet/lipgloss/blob/30441468/style.go#L214-L216) |
| Margins (all sides) | ✗ No | Explicitly excluded [style.go211-213](https://github.com/charmbracelet/lipgloss/blob/30441468/style.go#L211-L213) |
| Border Style | ✓ Yes | Border definitions inherit |
| Border Sides | ✓ Yes | Border visibility inherits |
| Transform | ✓ Yes | Function pointers can be inherited |

**Sources:** [style.go199-231](https://github.com/charmbracelet/lipgloss/blob/30441468/style.go#L199-L231)

The exclusion of padding and margins [style.go210-216](https://github.com/charmbracelet/lipgloss/blob/30441468/style.go#L210-L216)
 ensures that layout spacing remains explicit per style rather than accumulating through inheritance chains.
