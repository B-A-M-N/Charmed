Range-Based Styling
-------------------

### StyleRanges Function Architecture

The `StyleRanges` function provides segment-level styling control by applying different `Style` objects to specified text ranges.

The `StyleRanges` function processes text by stripping existing ANSI codes, applying styles to specified ranges using `ansi.Cut`, and reconstructing the final output with `strings.Builder`.

Sources: [ranges.go9-37](https://github.com/charmbracelet/lipgloss/blob/30441468/ranges.go#L9-L37)
 [ranges\_test.go9-110](https://github.com/charmbracelet/lipgloss/blob/30441468/ranges_test.go#L9-L110)

### Range Type Structure

The `Range` struct defines text segments for styling with precise index boundaries.

Range components:

*   `Start`: Inclusive starting index in the string
*   `End`: Exclusive ending index in the string
*   `Style`: The `Style` instance to apply to the range

The `NewRange` constructor creates `Range` instances for use with `StyleRanges`.

Sources: [ranges.go39-48](https://github.com/charmbracelet/lipgloss/blob/30441468/ranges.go#L39-L48)

### Using StyleRanges

The `StyleRanges` function allows you to apply different styles to different parts of a string. It takes a string and one or more Range objects, and returns a new string with the specified styles applied.

Example use cases:

| Scenario | Description |
| --- | --- |
| Syntax highlighting | Apply different styles to different parts of code or text |
| Search result highlighting | Highlight matching terms in search results |
| Interactive text | Style interactive elements differently |
| Rich text formatting | Create rich text with mixed styling |

Sources: [ranges.go9-37](https://github.com/charmbracelet/lipgloss/blob/30441468/ranges.go#L9-L37)
 [ranges\_test.go9-110](https://github.com/charmbracelet/lipgloss/blob/30441468/ranges_test.go#L9-L110)

Rune-Based Styling
------------------

### StyleRunes Function

The `StyleRunes` function provides character-level styling control by applying different styles to individual runes based on their indices.

The `StyleRunes` function converts strings to rune slices, creates index lookup maps, and applies matched/unmatched styles in grouped segments for optimal rendering.

Sources: [runes.go7-43](https://github.com/charmbracelet/lipgloss/blob/30441468/runes.go#L7-L43)
 [runes\_test.go8-64](https://github.com/charmbracelet/lipgloss/blob/30441468/runes_test.go#L8-L64)

### Index-Based Style Application

| Parameter | Type | Purpose |
| --- | --- | --- |
| `str` | `string` | Input text to style |
| `indices` | `[]int` | Rune positions to highlight |
| `matched` | `Style` | Style for runes at specified indices |
| `unmatched` | `Style` | Style for all other runes |

The function groups consecutive runes with the same style application to minimize ANSI escape sequence generation.

Sources: [runes.go10-43](https://github.com/charmbracelet/lipgloss/blob/30441468/runes.go#L10-L43)

### Unicode Handling

The `StyleRunes` function correctly handles Unicode characters including wide-width CJK characters:

| Test Case | Input | Indices | Result |
| --- | --- | --- | --- |
| ASCII | `"hello"` | `[0]` | Styles first character |
| CJK | `"你好"` | `[1]` | Styles second character |
| Mixed | `"hello 你好"` | `[6,7]` | Styles both CJK characters |

Sources: [runes\_test.go18-47](https://github.com/charmbracelet/lipgloss/blob/30441468/runes_test.go#L18-L47)

Advanced Style Combinations
---------------------------

The range styling system allows for sophisticated style combinations like:

### Multiple Non-Overlapping Ranges

Apply different styles to different parts of text:

    Input: "hello beautiful world"
    Ranges:
      - "hello" (Bold)
      - "world" (Italic)
    Output: "<bold>hello</bold> beautiful <italic>world</italic>"
    

Sources: [ranges\_test.go64-71](https://github.com/charmbracelet/lipgloss/blob/30441468/ranges_test.go#L64-L71)

    Input: "hello world"
    Ranges:
      - "hello" (Bold)
      - "world" (Italic) 
    Output: "<bold>hello</bold> <italic>world</italic>"

Use Cases\
---------\
\
Range-based styling is particularly useful for:\
\
1.  **Syntax highlighting**: Apply different styles to different parts of code\
2.  **Search highlighting**: Highlight search terms within longer text\
3.  **Interactive text interfaces**: Style interactive elements differently from static text\
4.  **Rich text representation**: Create complex rich text with multiple styles\
5.  **Terminal-based editors**: Apply formatting to specific regions of text\
\
The flexibility of range styling allows for complex visual representations in terminal applications while maintaining clean code structure.\
