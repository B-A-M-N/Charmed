Installation and Usage
======================

Relevant source files

*   [README.md](https://github.com/charmbracelet/glow/blob/752de97c/README.md?plain=1)
    
*   [main.go](https://github.com/charmbracelet/glow/blob/752de97c/main.go)
    

This document outlines the installation methods and basic usage patterns for Glow, a terminal-based markdown reader. It covers the various installation options, command-line interface (CLI) usage, terminal user interface (TUI) mode, and configuration settings. For information about Glow's architecture and internal systems, see [Architecture](https://deepwiki.com/charmbracelet/glow/2-architecture)
.

Installation
------------

Glow can be installed through multiple methods, including package managers, Go toolchain, or by building from source.

Sources: [README.md25-120](https://github.com/charmbracelet/glow/blob/752de97c/README.md?plain=1#L25-L120)

### Package Managers

Glow is available in several package managers:

| Platform | Command |
| --- | --- |
| macOS/Linux | `brew install glow` |
| macOS (MacPorts) | `sudo port install glow` |
| Arch Linux | `pacman -S glow` |
| Void Linux | `xbps-install -S glow` |
| FreeBSD | `pkg install glow` |
| Windows (Chocolatey) | `choco install glow` |
| Windows (Scoop) | `scoop install glow` |
| Windows (Winget) | `winget install charmbracelet.glow` |
| Debian/Ubuntu | See repository setup in README |
| Fedora/RHEL | See repository setup in README |

Sources: [README.md28-99](https://github.com/charmbracelet/glow/blob/752de97c/README.md?plain=1#L28-L99)

### Building from Source

Requirements:

*   Go 1.21 or later

Steps:

1.  Clone the repository: `git clone https://github.com/charmbracelet/glow.git`
2.  Change to the directory: `cd glow`
3.  Build the binary: `go build`

Sources: [README.md113-119](https://github.com/charmbracelet/glow/blob/752de97c/README.md?plain=1#L113-L119)

Command Line Arguments and Options
----------------------------------

Glow provides various command-line flags to customize its behavior:

Sources: [main.go396-416](https://github.com/charmbracelet/glow/blob/752de97c/main.go#L396-L416)
 [config\_cmd.go28-54](https://github.com/charmbracelet/glow/blob/752de97c/config_cmd.go#L28-L54)

### Key Flags

| Flag | Description | Default |
| --- | --- | --- |
| `-p, --pager` | Display with pager | `false` |
| `-t, --tui` | Display with TUI | `false` |
| `-s, --style` | Style name or JSON path | `auto` |
| `-w, --width` | Word-wrap at width (0 to disable) | `0` |
| `-a, --all` | Show system files and directories (TUI only) | `false` |
| `-l, --line-numbers` | Show line numbers (TUI only) | `false` |
| `-n, --preserve-new-lines` | Preserve newlines in output | `false` |
| `-m, --mouse` | Enable mouse wheel (TUI only) | `false` |
| `--config` | Config file path | Default location |

Sources: [main.go396-416](https://github.com/charmbracelet/glow/blob/752de97c/main.go#L396-L416)

TUI Navigation
--------------

In TUI mode, Glow provides an interactive interface with keyboard shortcuts for navigation.

| Key | Action |
| --- | --- |
| Arrow keys, h/j/k/l | Navigate through files and content |
| Enter | Open selected file |
| Esc, q | Go back or quit |
| /   | Search/filter files |
| ?   | Show help with all shortcuts |

Most of the keystrokes familiar from the `less` pager work in Glow's TUI mode.

Sources: [README.md130-132](https://github.com/charmbracelet/glow/blob/752de97c/README.md?plain=1#L130-L132)
