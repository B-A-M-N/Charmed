Project Dependencies
====================

Relevant source files

*   [go.mod](https://github.com/charmbracelet/soft-serve/blob/767bdd5a/go.mod)

This document provides an overview of the key external dependencies used by Soft Serve. Understanding these libraries and packages is essential for developers who wish to contribute to the project, extend its functionality, or understand its internal architecture.

Overview of Dependencies
------------------------

Soft Serve is built in Go and relies on a carefully selected set of external libraries to provide its core functionality. The project uses Go modules for dependency management, with all dependencies defined in the `go.mod` file.

Sources: [go.mod1-112](https://github.com/charmbracelet/soft-serve/blob/767bdd5a/go.mod#L1-L112)

Dependency Versioning and Management
------------------------------------

Soft Serve uses Go modules to manage dependencies, with versioning information maintained in the `go.mod` and `go.sum` files. The project is currently built with Go 1.23.0 and uses Go 1.24.1 as the toolchain.

Dependencies are categorized into:

1.  **Direct dependencies** - Listed at the top of the `go.mod` file
2.  **Indirect dependencies** - Dependencies of direct dependencies, listed in subsequent sections

Soft Serve adopts semantic versioning for its own releases and aims to use stable versions of dependencies where possible, though some dependencies (particularly those from the Charm ecosystem) might use specific Git commits where needed for cutting-edge features.

Sources: [go.mod3-5](https://github.com/charmbracelet/soft-serve/blob/767bdd5a/go.mod#L3-L5)
