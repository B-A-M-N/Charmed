This document details the various configuration options available in Glamour for customizing markdown rendering behavior. Glamour provides a rich set of options to control styling, layout, and rendering features when converting markdown to ANSI-styled terminal output. For information about the core rendering functions, see [Render Functions](https://deepwiki.com/charmbracelet/glamour/2.2-render-functions)
. For details about styling and themes, see [Styling System](https://deepwiki.com/charmbracelet/glamour/3-styling-system)
.

Configuration Overview
----------------------

Glamour provides configuration options via the `TermRendererOption` functional pattern. These options can be passed to `NewTermRenderer()` to customize the renderer's behavior.

Style Configuration Options
---------------------------

These options control the visual appearance of rendered markdown.

### Standard and Built-in Styles

Glamour includes several built-in styles that can be applied with a single option:

*   `WithStandardStyle(styleName string)` - Apply a built-in style by name:
    *   `"dark"` - Dark background theme
    *   `"light"` - Light background theme
    *   `"notty"` - Style for non-TTY environments
    *   `"ascii"` - ASCII-only style (no Unicode)
    *   `"dracula"` - Dracula-inspired theme
    *   `"pink"` - Pink-accented theme
    *   Several others available in the styles package
*   `WithAutoStyle()` - Automatically selects dark or light style based on terminal background

Sources: [glamour.go119-136](https://github.com/charmbracelet/glamour/blob/69661fd5/glamour.go#L119-L136)
 [glamour.go306-322](https://github.com/charmbracelet/glamour/blob/69661fd5/glamour.go#L306-L322)

### Custom Style Loading

Glamour provides multiple ways to apply custom styles:

*   `WithStylePath(stylePath string)` - Load style from a file path or fall back to built-in style
*   `WithStyles(styles ansi.StyleConfig)` - Apply styles directly from a StyleConfig object
*   `WithStylesFromJSONBytes(jsonBytes []byte)` - Parse and apply styles from JSON bytes
*   `WithStylesFromJSONFile(filename string)` - Load and apply styles from a JSON file

Sources: [glamour.go146-188](https://github.com/charmbracelet/glamour/blob/69661fd5/glamour.go#L146-L188)

### Environment-Based Configuration

*   `WithEnvironmentConfig()` - Use style specified by the `GLAMOUR_STYLE` environment variable

Sources: [glamour.go140-142](https://github.com/charmbracelet/glamour/blob/69661fd5/glamour.go#L140-L142)
 [glamour.go297-304](https://github.com/charmbracelet/glamour/blob/69661fd5/glamour.go#L297-L304)

Layout Configuration Options
----------------------------

These options control how content is formatted and wrapped.

*   `WithWordWrap(wordWrap int)` - Sets text wrapping width (default: 80 characters)
*   `WithTableWrap(tableWrap bool)` - Controls whether table content wraps:
    *   `true` (default) - Long cell content wraps within the cell
    *   `false` - Long cell content is truncated with an ellipsis
*   `WithInlineTableLinks(inlineTableLinks bool)` - Controls link rendering in tables:
    *   `true` - Links are rendered inline within table cells
    *   `false` (default) - Links are rendered as a list at the bottom of the table
*   `WithPreservedNewLines()` - Preserves newlines in content instead of replacing them

Sources: [glamour.go190-223](https://github.com/charmbracelet/glamour/blob/69661fd5/glamour.go#L190-L223)
 [ansi/renderer.go17-27](https://github.com/charmbracelet/glamour/blob/69661fd5/ansi/renderer.go#L17-L27)

Content Rendering Options
-------------------------

These options control how specific content features are rendered.

*   `WithBaseURL(baseURL string)` - Sets base URL for resolving relative links
*   `WithColorProfile(profile termenv.Profile)` - Sets color profile:
    *   `termenv.TrueColor` (default) - 24-bit color
    *   `termenv.ANSI256` - 256 colors
    *   `termenv.ANSI` - Basic 16 colors
*   `WithEmoji()` - Enables emoji rendering
*   `WithChromaFormatter(formatter string)` - Sets syntax highlighter formatter for code blocks

Sources: [glamour.go102-118](https://github.com/charmbracelet/glamour/blob/69661fd5/glamour.go#L102-L118)
