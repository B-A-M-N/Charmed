Example Application
===================

Relevant source files

*   [example/main.go](https://github.com/charmbracelet/fang/blob/d89b30af/example/main.go)
    

This document provides a detailed walkthrough of the complete example application located at [example/main.go1-154](https://github.com/charmbracelet/fang/blob/d89b30af/example/main.go#L1-L154)
 The example demonstrates fang's integration with Cobra commands, including command hierarchy, flag definitions, subcommands, signal handling, and error management. This application serves as both a functional demonstration and a reference implementation for building CLI applications with fang.

For information about the core components used in this example, see [Core Components](https://deepwiki.com/charmbracelet/fang/3-core-components)
. For configuration options, refer to [Configuration Options](https://deepwiki.com/charmbracelet/fang/4-configuration-options)
.

Core Test Scenarios
-------------------

### Help Rendering Tests

The help rendering system is validated through multiple scenarios that test different complexity levels and flag configurations.

#### Simple Command Help

The basic help scenario validates minimal CLI structure with essential commands and flags:

| Component | Validation |
| --- | --- |
| Usage Line | `simple [command] [--flags]` format |
| Commands Section | Completion and help commands presence |
| Flags Section | Help (`-h --help`) and version (`-v --version`) flags |
| Styling | Consistent spacing and section headers |

Sources: [testdata/TestSetup/simple/help.golden1-16](https://github.com/charmbracelet/fang/blob/d89b30af/testdata/TestSetup/simple/help.golden#L1-L16)

#### Complex Flags Help

The complex flags scenario tests comprehensive flag handling across different data types and configurations:

| Flag Type | Variations Tested |
| --- | --- |
| Boolean | Default false, default true, with shorthand |
| Integer | No default, with default, with shorthand |
| Float | No default, with default, with shorthand |
| String | With default value, no default, with shorthand |
| Special | `--no-help` flag handling |

Sources: [testdata/TestSetup/with\_flags/help.golden1-31](https://github.com/charmbracelet/fang/blob/d89b30af/testdata/TestSetup/with_flags/help.golden#L1-L31)

### Feature Integration Tests

These scenarios validate the injection and functionality of fang's enhanced features:

#### Completion System Tests

*   Shell completion script generation
*   Command and flag completion accuracy
*   Integration with cobra's completion system

#### Version Display Tests

*   Automatic version detection
*   Build information inclusion
*   Styled version output formatting

#### Man Page Generation Tests

*   `mango-cobra` integration functionality
*   ROFF format output validation
*   Command documentation completeness
