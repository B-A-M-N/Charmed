Purpose and Scope
-----------------

This document describes error handling and recovery mechanisms in Bubbletea. It covers built-in error types, panic recovery, context cancellation, and program termination scenarios. For information about testing error conditions, see [Testing Bubble Tea Applications](https://deepwiki.com/charmbracelet/bubbletea/5.4-testing-bubble-tea-applications)
. For signal handling configuration, see [Program Options](https://deepwiki.com/charmbracelet/bubbletea/2.5-program-options)
.

Built-in Error Types
--------------------

Bubbletea defines three sentinel errors that `Program.Run()` may return:

| Error | Description | When Returned |
| --- | --- | --- |
| `ErrProgramPanic` | Program recovered from a panic | A panic occurred in the event loop or command execution |
| `ErrProgramKilled` | Program was forcefully terminated | `Kill()` called, external context cancelled, or fatal error |
| `ErrInterrupted` | Program received interrupt signal | SIGINT received or `InterruptMsg` processed |

These errors are defined as package-level variables and can be checked using `errors.Is()`:

**Sources:** [tea.go29-37](https://github.com/charmbracelet/bubbletea/blob/f25595a8/tea.go#L29-L37)

### Recovery in Batch and Sequence Execution

Both `BatchMsg` and `sequenceMsg` have multi-layered panic recovery:

*   Outer defer in `execBatchMsg`/`execSequenceMsg`

Program Termination Scenarios
-----------------------------

### Termination Methods Comparison

| Method | Action | Terminal Cleanup | Final Render | Error Returned |
| --- | --- | --- | --- | --- |
| `Quit()` | Graceful exit | Full restoration | Yes | `nil` |
| `Kill()` | Immediate stop | Full restoration | No  | `ErrProgramKilled` |
| External context cancel | Immediate stop | Full restoration | No  | `ErrProgramKilled` + `context.Canceled` |
| `InterruptMsg` | Signal handler | Full restoration | No  | `ErrInterrupted` |
| Panic (recovered) | Emergency stop | Full restoration | No  | `ErrProgramKilled` + `ErrProgramPanic` |

**Sources:** [tea.go200-238](https://github.com/charmbracelet/bubbletea/blob/f25595a8/tea.go#L200-L238)
