This page lists and categorizes the major third-party dependencies declared in [go.mod1-156](https://github.com/charmbracelet/crush/blob/1ceeaa93/go.mod#L1-L156)
 It focuses on the **direct** dependencies that are explicitly imported by crush's own packages. Indirect dependencies are noted only when they are architecturally significant.

For information about how the build system consumes these dependencies, see page **8.1**. For details on the AI provider integration that many of these packages support, see page **4.2**.
