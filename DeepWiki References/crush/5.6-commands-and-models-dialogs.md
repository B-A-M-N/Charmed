This page documents the concrete dialog implementations in `internal/ui/dialog/`: Commands palette, Models picker, Sessions manager, API Key input, Quit confirmation, Reasoning selector, File picker (with image preview), Permissions dialog (with diff view), and OAuth dialogs. It also covers the `ListItem` interface and its implementations (`SessionItem`, `ModelItem`, `CommandItem`, `ReasoningItem`), and the `Action` type system.

For the base `Dialog` interface, `Overlay` stack, and `RenderContext` layout utility, see [Dialog System](https://deepwiki.com/charmbracelet/crush/5.5-dialog-system)
. For the `FilterableList` and `GroupedList` list components used inside dialogs, see [List Components](https://deepwiki.com/charmbracelet/crush/5.7-list-components)
.

---------------

`Commands` (ID: `"commands"`) is the command palette, opened with `ctrl+p`.

### Structure

The dialog holds a `FilterableList` with `CommandItem` entries and a `textinput.Model` for fuzzy filtering. It supports three tabs of commands: `SystemCommands`, `UserCommands`, and `MCPPrompts` [internal/ui/dialog/commands.go34-37](https://github.com/charmbracelet/crush/blob/1ceeaa93/internal/ui/dialog/commands.go#L34-L37)
