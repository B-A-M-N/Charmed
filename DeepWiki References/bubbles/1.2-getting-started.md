This page provides a quick start guide for integrating Bubbles v2 components into your Bubble Tea applications. Bubbles is a collection of reusable TUI components that implement the Model-View-Update (MVU) pattern.

Installation and Import Paths
-----------------------------

Bubbles v2 uses a new vanity import path. Ensure you are using Go 1.25.0 or later [go.mod1-3](https://github.com/charmbracelet/bubbles/blob/42130e89/go.mod#L1-L3)

### Import Migration

When upgrading from v1, you must update your import paths from `github.com/charmbracelet/bubbles/...` to `charm.land/bubbles/v2/...` [UPGRADE\_GUIDE\_V2.md38-77](https://github.com/charmbracelet/bubbles/blob/42130e89/UPGRADE_GUIDE_V2.md?plain=1#L38-L77)

| Package | v2 Import Path |
| --- | --- |
| **Core** | `charm.land/bubbles/v2` |
| **Inputs** | `charm.land/bubbles/v2/textinput`, `charm.land/bubbles/v2/textarea` |
| **Lists** | `charm.land/bubbles/v2/list` |
| **Navigation** | `charm.land/bubbles/v2/paginator`, `charm.land/bubbles/v2/filepicker` |
| **Status** | `charm.land/bubbles/v2/spinner`, `charm.land/bubbles/v2/progress` |
| **Utilities** | `charm.land/bubbles/v2/key`, `charm.land/bubbles/v2/help`, `charm.land/bubbles/v2/viewport` |

> \[!IMPORTANT\] Bubbles v2 requires **Bubble Tea v2** (`charm.land/bubbletea/v2`) and **Lip Gloss v2** (`charm.land/lipgloss/v2`) [UPGRADE\_GUIDE\_V2.md5-11](https://github.com/charmbracelet/bubbles/blob/42130e89/UPGRADE_GUIDE_V2.md?plain=1#L5-L11)

Key Changes in v2
-----------------

If you are migrating from v1, be aware of these structural changes:

1.  **Getters and Setters**: Many components moved from exported fields to methods. For example, use `m.SetWidth(40)` instead of `m.Width = 40` [UPGRADE\_GUIDE\_V2.md106-122](https://github.com/charmbracelet/bubbles/blob/42130e89/UPGRADE_GUIDE_V2.md?plain=1#L106-L122)
    
2.  **Key Messages**: Always use `tea.KeyPressMsg` for keyboard input handling [UPGRADE\_GUIDE\_V2.md94-104](https://github.com/charmbracelet/bubbles/blob/42130e89/UPGRADE_GUIDE_V2.md?plain=1#L94-L104)
    
3.  **Key Maps**: Global variables like `DefaultKeyMap` are now functions (e.g., `textinput.DefaultKeyMap()`) to prevent accidental global state mutation [UPGRADE\_GUIDE\_V2.md124-138](https://github.com/charmbracelet/bubbles/blob/42130e89/UPGRADE_GUIDE_V2.md?plain=1#L124-L138)
    
4.  **Styles**: Style initializers often require an `isDark bool` parameter now that `AdaptiveColor` has been removed from Lip Gloss v2 [UPGRADE\_GUIDE\_V2.md141-143](https://github.com/charmbracelet/bubbles/blob/42130e89/UPGRADE_GUIDE_V2.md?plain=1#L141-L143)
