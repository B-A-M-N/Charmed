Table
=====

The Table component in Bubbles provides a flexible way to display and interact with tabular data in terminal-based applications. It supports features like row selection, scrolling via an internal viewport, and customizable styling, making it ideal for presenting structured information in a readable format.

For other display components, see [Viewport](https://deepwiki.com/charmbracelet/bubbles/3.1-viewport)
 for scrollable content or [List](https://deepwiki.com/charmbracelet/bubbles/3.2-list)
 for selectable list items.

Architecture
------------

The Table component is built around several key concepts that work together to provide a complete tabular display solution. It uses an internal `viewport.Model` to manage the rendering area and scroll state.

### Core Concepts

Sources: [table/table.go16-29](https://github.com/charmbracelet/bubbles/blob/42130e89/table/table.go#L16-L29)
 [table/table.go31-38](https://github.com/charmbracelet/bubbles/blob/42130e89/table/table.go#L31-L38)
 [table/table.go42-51](https://github.com/charmbracelet/bubbles/blob/42130e89/table/table.go#L42-L51)
 [table/table.go106-110](https://github.com/charmbracelet/bubbles/blob/42130e89/table/table.go#L106-L110)

The table architecture consists of:

*   **Model**: The main state container [table/table.go16-29](https://github.com/charmbracelet/bubbles/blob/42130e89/table/table.go#L16-L29)
    
*   **Column**: Defines column headers and fixed widths [table/table.go35-38](https://github.com/charmbracelet/bubbles/blob/42130e89/table/table.go#L35-L38)
    
*   **Row**: A slice of strings representing data cells [table/table.go32](https://github.com/charmbracelet/bubbles/blob/42130e89/table/table.go#L32-L32)
    
*   **Styles**: Controls the visual appearance of headers, cells, and selected rows [table/table.go106-110](https://github.com/charmbracelet/bubbles/blob/42130e89/table/table.go#L106-L110)
    
*   **KeyMap**: Defines keyboard shortcuts for navigation, satisfying the `help.KeyMap` interface [table/table.go42-51](https://github.com/charmbracelet/bubbles/blob/42130e89/table/table.go#L42-L51)
    
*   **Viewport**: An internal `viewport.Model` that manages the scrollable area [table/table.go26](https://github.com/charmbracelet/bubbles/blob/42130e89/table/table.go#L26-L26)

Rendering Pipeline
------------------

The rendering of a table involves calculating visible rows based on the cursor position and joining cells horizontally using `lipgloss.JoinHorizontal`.

### Row Truncation and Joining

When a cell's content exceeds the `Column.Width`, the table uses `ansi.Truncate` to shorten the text and append an ellipsis [table/table.go296-302](https://github.com/charmbracelet/bubbles/blob/42130e89/table/table.go#L296-L302)

1.  **`View()`**: Combines `headersView()` and the internal `viewport.View()` [table/table.go251-253](https://github.com/charmbracelet/bubbles/blob/42130e89/table/table.go#L251-L253)
    
2.  **`UpdateViewport()`**: Calculates the `start` and `end` row indices based on the `cursor` and viewport height. It populates the viewport with rendered row strings [table/table.go264-283](https://github.com/charmbracelet/bubbles/blob/42130e89/table/table.go#L264-L283)
    
3.  **`renderRow(int)`**: Iterates through columns, applies styles (Cell or Selected), truncates content if necessary, and joins them into a single string [table/table.go286-312](https://github.com/charmbracelet/bubbles/blob/42130e89/table/table.go#L286-L312)

### Default Key Bindings

The `DefaultKeyMap` provides standard navigation [table/table.go67-102](https://github.com/charmbracelet/bubbles/blob/42130e89/table/table.go#L67-L102)
:

| Action | Keys | Implementation |
| --- | --- | --- |
| Line Up | `up`, `k` | `MoveUp(1)` |
| Line Down | `down`, `j` | `MoveDown(1)` |
| Page Up | `pgup`, `b` | `MoveUp(height)` |
| Page Down | `pgdown`, `f`, `space` | `MoveDown(height)` |
| Top | `home`, `g` | `GotoTop()` |
| Bottom | `end`, `G` | `GotoBottom()` |

Customization and Styles
------------------------

The `Styles` struct defines the look of the table [table/table.go106-110](https://github.com/charmbracelet/bubbles/blob/42130e89/table/table.go#L106-L110)
