Window Title and Colors
=======================

Purpose and Scope
-----------------

This document explains how Sequin processes and displays ANSI escape sequences related to window titles and terminal colors. These sequences allow terminal applications to modify the window title in the operating system's window manager and change terminal color settings. For information about text styling with SGR codes, see [SGR (Select Graphic Rendition)](https://deepwiki.com/charmbracelet/sequin/3.1-sgr-(select-graphic-rendition))
.

Window Title Commands
---------------------

Sequin handles Operating System Command (OSC) sequences that allow applications to set the terminal window's title and icon name. These commands are commonly used by terminal applications to provide context about the running application or current directory.

### Supported Title Commands

| OSC Command | Purpose | Example Output |
| --- | --- | --- |
| OSC 0 | Set both icon name and window title | `Set icon name and window title to "terminal"` |
| OSC 1 | Set icon name only | `Set icon name to "terminal"` |
| OSC 2 | Set window title only | `Set window title to "hello"` |

The title command takes the format `OSC n;title`, where `n` is the command number (0, 1, or 2) and `title` is the text to set as the title.

Terminal Color Commands
-----------------------

Sequin also processes OSC sequences that control terminal colors. These commands allow applications to modify the default colors used by the terminal.

### Setting and Requesting Colors

The terminal color commands support two operations:

1.  Setting a color value
2.  Requesting the current color (by using `?`)

| OSC Command | Purpose | Example Output |
| --- | --- | --- |
| OSC 10;_value_ | Set foreground color | `Set foreground color` |
| OSC 11;_value_ | Set background color | `Set background color` |
| OSC 12;_value_ | Set cursor color | `Set cursor color` |
| OSC 10;? | Request foreground color | `Request foreground color` |
| OSC 11;? | Request background color | `Request background color` |
| OSC 12;? | Request cursor color | `Request cursor color` |

### Resetting Colors

Terminal colors can be reset to their default values using the following commands:

| OSC Command | Purpose | Example Output |
| --- | --- | --- |
| OSC 110 | Reset foreground color | `Reset foreground color` |
| OSC 111 | Reset background color | `Reset background color` |
| OSC 112 | Reset cursor color | `Reset cursor color` |

Sources: [color.go10-56](https://github.com/charmbracelet/sequin/blob/025641c1/color.go#L10-L56)

Implementation Details
----------------------

Sequin processes window title and color commands by intercepting OSC sequences and parsing their parameters to generate human-readable explanations.

### Code Flow for Title Commands

The `handleTitle` function processes window title commands with these steps:

1.  Split the OSC data on semicolons to extract command parameters
2.  Check that exactly two parts are present (command and title)
3.  Based on the OSC command (0, 1, or 2), format an appropriate explanation
4.  Return the formatted explanation or an error

Sources: [title.go11-26](https://github.com/charmbracelet/sequin/blob/025641c1/title.go#L11-L26)

### Code Flow for Color Commands

The `handleTerminalColor` function handles setting or requesting terminal colors:

1.  Split the OSC data on semicolons to extract command parameters
2.  Determine if setting or requesting a color (value vs "?")
3.  Based on the OSC command (10, 11, or 12), format an explanation for the appropriate color type
4.  Return the formatted explanation or an error

The `handleResetTerminalColor` function processes reset commands with similar logic but for OSC 110, 111, and 112.

Sources: [color.go10-56](https://github.com/charmbracelet/sequin/blob/025641c1/color.go#L10-L56)

Example Usage
-------------

To see how Sequin displays window title and color commands, you can use it to process terminal output containing these sequences:

    $ echo -e "\033]0;My Window Title\007" | sequin
     OSC 0;My Window Title: Set icon name and window title to "My Window Title"
    
    $ echo -e "\033]2;Terminal\007" | sequin
     OSC 2;Terminal: Set window title to "Terminal"
    
    $ echo -e "\033]10;#336699\007" | sequin
     OSC 10;#336699: Set foreground color
    

These examples show how Sequin translates the raw OSC sequences into human-readable explanations, making it easier to understand what's happening in the terminal.
