This page provides a comprehensive guide to installing Sequin across various platforms and package managers. Sequin is available through multiple distribution channels to accommodate different operating systems and user preferences.

Overview of Installation Methods
--------------------------------

Sequin can be installed in several ways:

1.  Through package managers (Homebrew, Arch Linux AUR, Nix, apt, yum)
2.  By downloading pre-built binaries or packages
3.  Using Go's package manager
4.  Building from source

Sources: [README.md22-77](https://github.com/charmbracelet/sequin/blob/025641c1/README.md?plain=1#L22-L77)

Direct Downloads
----------------

Pre-built binaries and packages are available from the [GitHub Releases page](https://github.com/charmbracelet/sequin/blob/025641c1/GitHub%20Releases%20page)

*   **Packages**: Available in Debian (.deb) and RPM formats
*   **Binaries**: Available for Linux, macOS, Windows, FreeBSD, OpenBSD, and NetBSD

Simply download the appropriate file for your platform and follow the standard installation procedure for your operating system.

Sources: [README.md66-71](https://github.com/charmbracelet/sequin/blob/025641c1/README.md?plain=1#L66-L71)
