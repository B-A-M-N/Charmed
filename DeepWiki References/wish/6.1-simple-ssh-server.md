Basic Server Setup
------------------

Creating a simple SSH server with wish involves a few straightforward steps. At its core, you'll use the `wish.NewServer()` function with appropriate options to configure your server.

Let's look at the essential components of a basic SSH server:

### Server Creation and Configuration

The `wish.NewServer()` function accepts various configuration options:

| Option | Description |
| --- | --- |
| `WithAddress()` | Sets the network address (host:port) the server will listen on |
| `WithHostKeyPath()` | Specifies the path to the SSH host key |
| `WithMiddleware()` | Adds middleware functions to the server |

Sources: [examples/simple/main.go19-41](https://github.com/charmbracelet/wish/blob/19f43236/examples/simple/main.go#L19-L41)

Graceful Shutdown
-----------------

For production applications, implementing graceful shutdown is important. This allows your server to finish handling current connections before shutting down.

A graceful shutdown pattern typically involves:

1.  Starting the server in a goroutine
2.  Setting up a signal handler
3.  Using `Shutdown()` with a context to gracefully stop the server

Summary
-------

Creating a simple SSH server with wish involves:

1.  Creating a server with `wish.NewServer()`
2.  Configuring options like address and host key
3.  Adding middleware for functionality
4.  Starting the server with `ListenAndServe()`
5.  (Optional) Implementing graceful shutdown

This pattern provides a foundation for more complex SSH applications, including TUI applications via BubbleTea, Git servers, and file transfer services.
